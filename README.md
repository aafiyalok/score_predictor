# 🎓 Student Score Predictor (Machine Learning)

## 📌 Overview

This project predicts student scores based on the number of study hours using a Machine Learning model.

## 🚀 Features

* Data visualization using Matplotlib
* Linear Regression model using Scikit-learn
* Prediction on new data
* Model evaluation using Mean Absolute Error

## 🛠️ Tech Stack

* Python
* Pandas
* Matplotlib
* Scikit-learn

## 📊 How it works

1. Data is created and loaded
2. Dataset is split into training and testing sets
3. Linear Regression model is trained
4. Predictions are made and evaluated

## ▶️ Run the project

```bash
pip install -r requirements.txt
python student_score.py
```

## 📈 Output

* Displays prediction results
* Shows regression graph
